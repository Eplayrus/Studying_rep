Документация для проекта:

Описание проекта

Проект представляет собой реализацию Twitter API на основе Flask. Возможности:

Создание новых твитов
Изменение существующих твитов
Удаление твитов
Чтение списка всех твитов
Классы и методы

Twit
__init__(self, body, author): Инициализирует объект класса Twit с телом (body) и автором (author).

to_dict(self): Возвращает словарь с представлением твитов в JSON-формате.

User (не используется)
__init__(self, userName): Инициализирует объект класса User с логином (userName).

Функции
/ping
GET-метод: Проверяет доступность сервера. Возвращает ответ "pong".
/twit
POST-метод: Создает новый твит. Ожидаемые данные: body, author. Возвращает ответ с success, если твит создан успешно.
PATCH-метод: Изменяет существующий твит. Ожидаемые данные: body, author. Удаляет последний добавленный твит и возвращает ответ с success, если изменения произведены успешно.
DELETE-метод: Удаляет последний добавленный твит. Возвращает ответ с success, если удаление произведено успешно.
GET-метод: Возвращает список всех твитов в JSON-формате.

Инструкция по использованию API Twitter.

**Создание нового твитта**

*   **Метод:** POST
*   **URL:** `/twit`
*   **Тело запроса:**
    ```json
{
  "body": "Сообщение о создании нового твитта",
  "author": "Имя автора"
}
```
*   Пример ответа:
    ```json
{
  "response": "success"
}
```

**Чтение всех твиттов**

*   **Метод:** GET
*   **URL:** `/twit`
*   Возвращается список всех созданных твиттов.

Примечание: в случае если класс `Twit` не может быть сериализован в JSON, необходимо доработать этот класс, чтобы он удовлетворял требованиям сериализации в JSON.
